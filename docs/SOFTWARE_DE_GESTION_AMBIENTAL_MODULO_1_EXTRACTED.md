INTRODUCCIÓN

El proyecto consiste en el desarrollo de un software de gestión ambiental, documental y educativa, diseñado para ayudar a empresas a cumplir normativas ambientales y de calidad (ISO 14001 e ISO 9001), automatizando procesos como generación de documentos, control de procedimientos y capacitación del personal.
